# At-Bay Cyber Defenders

A browser-based 2D platformer game where you fight cyber threats as a At-Bay cyber security hero.

**Play now:** https://idant-at-bay.github.io/cyber-intro/

## How to Play

Open `index.html` in any modern web browser, or visit the link above to play online.

### Controls

| Key | Action |
|-----|--------|
| <- -> Arrow Keys / AD | Move left/right |
| W / Up Arrow | Jump (double jump enabled) |
| Space | Shoot |

### Heroes

Choose from 7 different heroes, each with their own unique style:

- **Tamir** - Feature Generator
- **Gal** - Refactor Master
- **Chen** - MVP Conductor
- **Yishay** - MSOD Fighter
- **Idan** - UV Migrator
- **Sali** - Captain PR
- **Benny** - VP of Cyber

### Bosses

Defeat 3 cybersecurity-themed bosses across 3 levels:

1. **MALWARE.COM** - The Corrupted Domain
2. **PHISHING MAIL** - The Deceptive Message
3. **FAKE SECURITY** - The False Report Service

Each boss has unique attack patterns and increases in difficulty. Bosses become more aggressive as their health decreases.
